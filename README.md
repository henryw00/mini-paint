# Mini-Paint

An AI-powered digital sandbox for testing and generating miniature paint schemes. 

## Overview
Army color schemes are deeply personal and people spend a lot of time ideating exactly what they want their 
minis to look like/how to make them stand out. Mini paint helps bring ideas to life and allow people to 
experiment digitally before any physical painting occurs. 

## Architecture
* **Frontend:** React + Konva.js (Canvas rendering)
* **Core API:** Java 
* **ML Service:** Python + FastSAM (Semantic Segmentation)
